# God Money Vault (Obsidian)

Esta pasta foi organizada para funcionar como um **vault do Obsidian** para o projeto **God Money**.

## Como usar

1. Abra o **Obsidian**
2. Clique em **Open folder as vault**
3. Selecione esta pasta:

```text
D:\jvictor doc\edicao\Antigravity\god-money-bot\docs\obsidian
```

## Objetivo

Este vault serve para organizar o projeto sem depender do Obsidian para o bot funcionar.

Use o Obsidian para:

- estratégia do produto
- documentação dos comandos
- fontes de notícias
- checklist de deploy
- organização do Supabase
- operação do bot
- ideias de conteúdo e melhorias

## Estrutura

- `00-Home/` → visão geral do projeto
- `01-Estrategia/` → posicionamento e objetivo
- `02-Comandos/` → regras dos comandos do Telegram
- `03-Fontes/` → fontes, scraping e observações
- `04-Deploy/` → Render, ambiente e checklist
- `05-Supabase/` → banco, schema e queries
- `06-Operacao/` → bugs, tarefas e manutenção
- `07-Conteudo/` → mensagens, roteiros e textos

## Observação

O bot continua rodando normalmente em Node.js + Render + Supabase.
O Obsidian aqui é sua **central de organização**.
