# Funnel de Monetização

## Etapa 1 — Aquisição

Entrada por:

- canal Telegram
- Instagram / Reels
- conteúdo financeiro curto
- comunidades de bolsa
- indicação de usuários

Objetivo:

- fazer o usuário entrar no bot

## Etapa 2 — Ativação

Primeiros 3 minutos do usuário:

1. `/start`
2. escolher perfil
3. montar watchlist inicial
4. receber primeiro resumo útil

Meta:

- fazer o usuário perceber valor imediatamente

## Etapa 3 — Retenção

Mecanismos:

- resumo pré-mercado
- resumo de fechamento
- alertas por watchlist
- agenda econômica personalizada

Meta:

- transformar uso em hábito

## Etapa 4 — Conversão

Triggers de upgrade:

- limite de 3 ativos no free
- alerta premium bloqueado
- análise premium travada
- feed personalizado parcial

Mensagem ideal:

"Você já acompanha 3 ativos. No Pro, desbloqueie sua watchlist completa e alertas personalizados."

## Etapa 5 — Expansão de receita

- upsell para Premium
- comunidade VIP
- relatórios avulsos
- afiliados

## Métricas principais do funil

- start → ativação
- ativação → usuário recorrente 7 dias
- retenção 7d / 30d
- conversão Free → Pro
- ARPU
- LTV
- churn
